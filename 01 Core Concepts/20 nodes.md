- A Kubernetes cluster is made up of [[19 Control Plane|control plane nodes]] and [[22 worker node|worker nodes]].
- The scheduler places workloads on nodes, while the [[08 kubelet]] and [[09 kube-proxy]] handle node-level operations.
- Start with [[00 Cluster Architecture]] if you want the high-level picture.
