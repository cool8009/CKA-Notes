- Worker nodes run application workloads in [[10 POD]]s.
- Each worker node runs the [[08 kubelet]] and [[09 kube-proxy]], and commonly hosts add-ons delivered as a [[8 DaemonSets|DaemonSet]].
- See [[20 nodes]] for the broader node model and [[10 Static Pods]] for kubelet-managed pods.
